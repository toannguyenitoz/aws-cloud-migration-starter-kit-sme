# Contributing

This is a learning repository. Contributions should keep the content practical, beginner-friendly, and safe.

## Rules

- Do not commit secrets.
- Do not include real customer data.
- Keep examples generic.
- Add warnings when resources may create cost.
- Prefer clear explanations over complex architecture.
